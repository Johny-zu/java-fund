# Encarpetado BD
Encarpetado que sirve para registrar las esquemas de las BD para probar con lo que se esta trabajando.

## Esquema personas
En este se ingreso los comandos utilizados para crear la BD de crud personas para revisar que es lo que contine y como hacer algunas pruebas para revision de actividad

## Esquema hotel
En este se ingresa los comandos utilizados para el ingresa del esquema de la gestion hotelera con sus diferentes clases tablas y pruebas unitarias solo para conocer el estado de la BD
