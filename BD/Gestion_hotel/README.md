# Informacion de la actividad
Un hotel pequeño necesita un sistema para gestionar sus habitaciones, huéspedes, reservas y pagos. Cada reserva está asociada a un huésped y una o varias habitaciones por un período de fechas. El sistema debe verificar disponibilidad y calcular costos automáticamente.

Para realizar la siguiente actividad se estara realizando con la siguiente informacion para dar una estructura mas clara de lo que se esta realizando

## Encarpetado Modelo
En este encarpetado hallaremos las clases objetos con las que se estaran trabajando las cuales seran las siguientes: 
 - Habitacion
 - Huesped
 - Reserva
 - Detalles para reserva (Representara la habitacion detro de una reserva)
 - Pago

Estas se hacen con el fin de dar las clases que se usaran de molde para todas las actividades que se estaran trabajando, y estas se basan en la estructura del esquema de la BD.
