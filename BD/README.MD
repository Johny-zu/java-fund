# Ingreso de informacion sobre ejercicio de BD
Para el inicio de este encarpetado se ingreso a la pagina oficial de mysql y se descargo de las siguientes legas lo necesario:
 - Aqui para descargar el motor de mysql: https://dev.mysql.com/downloads/file/?id=549397
 - Se descarga el cliente de mysql para el uso grafico: https://dev.mysql.com/downloads/workbench/

Con esto ya descargado se ingresaran los datos para las tablas de cada ejercicio.

Para empezar se usara una clase .sql con el fin de tener respaldada las cosas que se tengan, empezaremos con algo sencillo para dar a conocer como funciona el crud y como conectarnos a una BD

Se realizara un encarpetado para el control del ejercicio y que se tenga todo en orden. Ademas se ingresara una nueva carpeta para el ingreso del archivo SQL
